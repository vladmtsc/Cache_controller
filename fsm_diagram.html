Cache Controller FSM

4-Way Set-Associative Cache  |  Write-Back + Write-Allocate  |  LRU Replacement

IDLE – Waiting for CPU request
READ_HIT – Return cached word to CPU
READ_MISS – Fetch block from memory
WRITE_HIT – Write word, mark dirty
WRITE_MISS – Fetch + merge CPU write, mark dirty
EVICT – Writeback dirty LRU block to memory
